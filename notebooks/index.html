These Jupyter notebooks provide practical implementations of the concepts covered in the training sessions. Each notebook includes detailed explanations, code examples, and exercises to reinforce your understanding.

To get started, download the notebooks and run them in your local Jupyter environment or use a cloud-based notebook service.

Requirements

  • Python 3.8+
  • Jupyter Notebook or JupyterLab
  • Required packages are listed in each notebook's introduction

You can install the common dependencies using:

pip install -r requirements.txt
